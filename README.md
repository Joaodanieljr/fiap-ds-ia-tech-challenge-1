# Tech Challenge 1: Análise e Predição de NPS 📊

> Projeto desenvolvido para o curso de Data Science e IA da FIAP, focado em resolver problemas de satisfação do cliente através de análise de dados.

## 📝 Sobre o Projeto
O objetivo deste desafio é analisar dados de clientes para entender os fatores que influenciam o **NPS (Net Promoter Score)** e desenvolver um modelo capaz de prever a classificação de um cliente (Promotor, Neutro ou Detrator).

A solução engloba desde a limpeza e exploração dos dados (EDA) até o treinamento de algoritmos de Machine Learning para auxiliar na tomada de decisão estratégica.

## 🚀 Funcionalidades
- **Análise Exploratória (EDA):** Identificação de padrões e correlações nos dados.
- **Engenharia de Features:** Tratamento de variáveis para melhorar a performance do modelo.
- **Modelagem Preditiva:** Treinamento de modelos para classificação de satisfação.
- **Avaliação de Métricas:** Uso de Precision, Recall e F1-Score para validar os resultados.

## 🛠 Tecnologias Utilizadas
- **Python** (Linguagem principal)
- **Pandas & NumPy** (Manipulação de dados)
- **Scikit-Learn** (Machine Learning)
- **Matplotlib & Seaborn** (Visualização de dados)
- **Jupyter Notebook** (Ambiente de desenvolvimento)

## 📁 Estrutura do Repositório
- `notebooks/`: Contém os arquivos `.ipynb` com a análise completa.
- `data/`: Base de dados utilizada no desafio.
- `requirements.txt`: Lista de dependências do projeto.

## 🔧 Como Rodar o Projeto
1. Clone este repositório:
   ```bash
   git clone [https://github.com/Joaodanieljr/fiap-ds-ia-tech-challenge-1.git](https://github.com/Joaodanieljr/fiap-ds-ia-tech-challenge-1.git)
   ```

2. Crie um ambiente virtual (opcional, mas recomendado):
   ```bash
   python -m venv venv
   source venv/bin/activate  # No Windows: venv\Scripts\activate
   ```

3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

4. Abra o Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
